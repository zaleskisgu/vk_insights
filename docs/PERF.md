# PERF

Lighthouse после `npm run build`, поднятое приложение, профиль как в отчёте (обычно Mobile).

## Замер 1

![Lighthouse](./images/PERF_photo_1.png)

| Метрика | Значение | Lighthouse |
|---------|----------|------------|
| Performance | 80 | 50–89 |
| LCP | 3,0 s | fail — основной вклад в балл |
| FCP | 1,1 s | 50–89 |
| Speed Index | 1,4 s | 50–89 |
| TBT | 30 ms | pass |

**LCP:** узкое место — крупный контент первого экрана.

**TBT:** занижен из‑за **прогретого кэша на бэкенде** (меньше сети/ожидания); на холодном API TBT может быть выше.
